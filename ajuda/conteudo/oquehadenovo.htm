O que há de novo

Esta versão do Cliente do Prisma possui vários novos recursos. Estes recursos estão listados abaixo:

Versão 1.0

  • Instalador Único. Agora é possível instalar o cliente do Prisma em poucos cliques.
  • Independência do Java Runtime. Não é mais necessário instalar o Java antes de configurar a geração de arquivos em PDF.
  • Menus do Accuterm em Português (Brasil). Os menus do emulador Accuterm foram traduzidos para a língua portuguesa.
  • Menu Ir Para. O Accuterm possui um menu com atalhos rápidos para alguns sistemas do INSS. Nele será possível acessar consultas a CEP, micro-regiões, GET, CNIS, SIBE, dentre outros.
  • Colar Texto Inteligente. Devido a incapacidade do Prisma de processar sinais e caracteres acentuados, a colagem de texto foi modificada para traduzir textos incompatíveis com o Prisma. Ao colar um texto com acentos, estes serão removidos. Ex: concessão será traduzido para concessao. E 123.456.789-00 será traduzido para 12345678900.
  • Gerador de Atalhos Para Máquinas Prisma. Agora é possível gerar múltiplos atalhos na área de trabalho para diferentes máquinas Prisma facilmente através de um assistente.
  • Temas. É possível escolher diferentes cores para o Prisma de acordo com a preferência do usuário.
  • Geração de Arquivos PDF. O componente que transforma as impressões do Prisma em arquivos PDF é instalado automaticamente.
  • Integração com PDF24. É possível configurar a geração de arquivos PDF para enviar o relatório diretamente para o aplicativo PDF24, sem o uso de intermediários.
  • Limpeza de Lixo Automática. O lixo de arquivos gerados durante a produção de arquivos PDF é excluído automaticamente pelo Windows.
