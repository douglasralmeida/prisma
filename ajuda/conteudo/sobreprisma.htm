Sobre o Prisma

Prisma é o sistema desenvolvido pela Dataprev e utilizado pelo INSS para análise de direito previdenciário.

O sistema fica armazenado em servidores na rede de computadores do INSS que são denominadas Máquinas Prisma.

Para acessar o Prisma, é necessário um aplicativo que emula um terminal de acesso. No INSS, é utilizado o aplicativo Accuterm.
